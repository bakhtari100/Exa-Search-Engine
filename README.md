# TikTok Hashtag Trend Analyzer
A Python tool that tracks how TikTok hashtags evolve from 2022 to 2025 for any topic you search. It uses Exa’s search API to find TikTok posts, extracts hashtags, performs sentiment analysis, and visualizes top hashtag trends over time.

## ***Features***
* Search any topic and collect TikTok hashtags
* Analyze sentiment (positive, negative, neutral)
* View top hashtags per year
* Plot hashtag frequency trends (2022–2025)

## ***Requirements***
* Python 3
* exa-py
* textblob
* matplotlib


## ***Install with:***

pip install exa-py textblob matplotlib

## ***How to Run:***

Enter a topic when prompted (e.g., summer fashion). The script will:
1. Print top hashtags per year
2. Show sentiment of posts
3. Display a line chart of hashtag trends
